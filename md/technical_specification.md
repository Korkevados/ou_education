# אפיון טכני לאתר OU

## Frontend

- **Framework**: Next.js
- **UI Library**: Shadcn/ui
- **Styling**: Tailwind CSS

## Backend

- **Database**: Supabase
- **Authentication**: Supabase Auth (כולל אימות דו-שלבי עם SMS)

## שירותים חיצוניים

- **SMS**: שירות 019 לשליחת הודעות SMS.
- **Videos**: הטמעת סרטוני YouTube.
